Developed and maintained by **Nhial Cham** — Penetration Tester, Full-Stack Developer, Data Analyst, and Leader of the University of Juba Cyber Security Club.

---

## ⚡ Core Operational Features

* **Interactive Terminal Shell:** A functional, sandboxed CLI parser implementing customized input string matching for structural parameters (`help`, `whoami`, `skills`, `status`, `clear`).
* **Live Packet Hub Monitor:** An automated background telemetry logging stream simulating real-time firewall block rates and network intrusion events.
* **Matrix Capability Matrix:** A visual efficiency tracking array mapping defensive capabilities, offensive vectors, and programming paradigms.
* **Case File Registry:** Classified secure dossier layouts detailing active engineering projects, full-stack database engines, and local data disaggregation studies.
* **CRT System Overlay:** Purely native CSS processing engine providing old-school Cathode-Ray Tube scanlines and responsive matrix color profiles without reliance on heavy external visual frameworks.

---

## 🛠️ Tech Stack & Architecture

* **Core Logic:** React 18 (Declarative UI state mapping)
* **Compilation/Bundling:** Vite 5 (Native browser ES module routing)
* **Styling & FX:** Vanilla CSS3 (Custom keyframe animations & pulse loops)
* **Typography:** Fira Code Monospace (Engineering optimized ligatures)
